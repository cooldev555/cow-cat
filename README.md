## Requirements

- **PHP : 5.6**
- **Laravel : 5.1.***
- **Composer**

## My Development Environment

- Homebrew
- PHP - 7.0.7
- MySQL - 5.7.13
- Redis - 3.2.0
- Nginx - 1.10.1

## Installation

1. **`git clone https://github.com/luisedware/CowCat.git`**
1. **`cd CowCat`**
1. **`sudo composer install`**
1. **`sudo npm install`**
1. **`sudo vi .env`**
1. **`php artisan migrate:refresh --seed`**
1. **`php artisan serve`**
1. **`gulp watch`**
